## Как пользоваться

### Установка зависимостей

```bash
$ npm install
```

### Проверка кода

```bash
$ npm run lint
```

### Запуск всех тестов

```bash
$ npm run test

```

### Запуск в Dev режиме

```bash
$ npm run start
```

### Запуск в Prod режиме

```bash
$ npm run build
```
